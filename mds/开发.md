## Node.js 的使用

遍历一个文件夹，筛选出特定类型的文件

```javascript
let fileList = fs.readdirSync(filePath);
fileList = fileList.filter((file) => {
  const extname = path.extname(file).toLowerCase();
  return extname === '.md' || extname === '.markdown';
});
```
